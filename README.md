🍔 Food Delivery System (React + Django)
📌 Project Overview

This is a full-stack Food Delivery System built using React (Frontend) and Django REST Framework (Backend).
The system is similar to Foodpanda, where users can create accounts, browse food items, and place orders, while delivery riders can manage and deliver orders.

🚀 Features
👤 User Side
User registration and login
Browse food items
Add items to cart
Place orders
View order history
🛵 Rider Side
Rider registration and login
View assigned delivery orders
Update delivery status (picked, on the way, delivered)
🧑‍💼 Admin Side
Manage users and riders
Add/update/delete food items
Manage orders
Track delivery system
🛠️ Tech Stack
Frontend:
React.js
Tailwind CSS
JavaScript (ES6+)
Fetch API / Axios
HTML
Backend:
Django
Django REST Framework (DRF)
SQLite / PostgreSQL
CORS Headers
🔗 API Structure
/api/users/ → User management
/api/riders/ → Rider management
/api/foods/ → Food items
/api/orders/ → Order system
/api/auth/ → Authentication
⚙️ How It Works
Users and riders create accounts separately
Users browse food items and place orders
Orders are assigned to riders
Riders update delivery status
Admin manages the whole system via dashboard
🎨 Frontend UI
Built with React + Tailwind CSS
Fully responsive design (mobile + desktop)
Clean and modern UI similar to Foodpanda
📦 Installation
Backend (Django)
pip install -r requirements.txt
python manage.py runserver
Frontend (React)
npm install
npm run dev
